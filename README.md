# Used car selling price prediction model

A Machine Learning project created to predict the resale value of used vehicles based on a kaggle dataset. This project explores the transition from a single **Decision Tree** to a **Random Forest Ensemble** to achieve higher predictive accuracy.

Data has been split into two pieces: the **training set** has been used to fit the model, whilst the **validation set** has been used to compare our predicted selling prices to the actual ones and calculate the **Mean Absolute Error**.

Results: **MAE $\approx$ 715.7**
